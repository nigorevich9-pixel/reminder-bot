# reminder-bot — roadmap

## Реализовано

- UI для core (events + команды run/hold/ask)
- напоминания (due reminders loop)
- базовые уведомления по core task'ам
- опциональная Jira интеграция

## Не хватает / желательно

- Выбор проекта/репозитория в `/core` (сейчас `project_id` в events всегда null).
- Более понятные тексты/UX вокруг `/hold` (см. known_issues).
- Режим “просмотр задач” (list, filters) для удобства пользователя.
