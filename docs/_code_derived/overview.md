# reminder-bot — обзор

Репо содержит две подсистемы:

1) **Telegram UI для core**
- диалог `/core` (question/task)
- команды `/run`, `/hold`, `/ask`
- запись событий в `events`
- уведомления пользователю по статусам `tasks`

2) **Напоминания и интеграции**
- CRUD напоминаний (таблица `reminders`)
- отправка напоминаний по расписанию
- (опционально) Jira интеграция

## Граница

Core-таски и напоминания — разные сущности. Core **не** является “хранилищем напоминаний”.
